# Multi Launch Search Provider

A GNOME Shell extension that adds a search provider to the Overview, allowing users to launch multiple applications at once by separating their names with a semicolon (`;`).

**Author:** Sobeitnow

## âœ¨ Features

- **Batch Launching:** Open multiple apps in a specific order with a single command.
- **Fuzzy Search:** Detects installed applications even if you type only part of the name (e.g., `fire; calc` opens Firefox and Calculator).
- **Native Integration:** Appears seamlessly in the GNOME Overview search results.

## ðŸš€ Usage

1. Press the **Super** key (Windows key) to open the Overview.
2. Type the names of the applications you want to open, separated by a semicolon `;`.
   * *Example:* `firefox ; terminal ; files`
3. A result named "Multi Launch" will appear.
4. Press **Enter** to launch all of them.
